# BillZap ⚡

Free GST Billing App for Indian MSMEs.

## Features
- Create GST invoices (CGST / SGST / IGST)
- PDF generation & WhatsApp share
- Customer & Product catalog
- Expense tracker & P&L
- 100% Offline — no internet required
- No Firebase — all data stored locally on device

## Run
```bash
flutter pub get
flutter run
```
